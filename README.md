# 🧮 Modern Calculator

Welcome to my Modern Calculator project! This calculator is built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean, modern, and responsive user interface.

---

## 🚀 About

This project is designed and developed to demonstrate my front-end web development skills. It features a beautiful glassmorphism design, smooth animations, responsive layout, and fully functional calculator operations.

---

## ✨ Features

- 🎨 Modern Glassmorphism UI
- 📱 Fully Responsive Design
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 💯 Percentage Calculation
- 🔢 Decimal Support
- ⌨️ Keyboard Support
- 🗑️ Delete (DEL) Button
- 🧹 All Clear (AC) Button
- ⚡ Fast and Smooth Performance
- 🌙 Attractive Dark Theme
- 🎯 User-Friendly Interface

---

## ▶️ How to Run

1. Download or clone this repository.
2. Open the `calculator` folder in Visual Studio Code.
3. Open `index.html` using Live Server or any web browser.
4. Start performing calculations.
